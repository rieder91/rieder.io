Thomas Rieder's Personal Website
========

HTML, CSS and Javascript sources for my personal website

# Dependencies

```
# you need to install bower first
npm i -g bower
npm i -g grunt-cli
```

# Development

```
# install libraries
npm install
bower install

# build
grunt

# start a webserver (python3)
python -m http.server 3000

# --> head to http://localhost:3000
```
