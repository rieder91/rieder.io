My Interests

Photography

In my spare time I do a lot of photography. I focus on landscape, macro and wildlife shots. This is why a wide angle and telephoto lens are my best friends. When travelling I'm almost always taking my gear with me - although it's quite heavy. I'm currently using a Canon 7D with four lenses: wide angle (10-22mm), general purpose (17-55mm), macro (100mm) and telephoto (70-300mm). Given the limitations of a crop sensor I plan to eventually upgrade to full-frame. You can check out some of my shots on Flickr.

Security

I really enjoy analyzing code. Of course there are a lot of vulnerabilities with web applications (first and foremost injection attacks) but I am more passionate about overflow attacks. I find the low-level nature of buffer/heap overflows or return-to-libc attacks more pure. I also took part in several CTFs and the CCC is usually a must for me. Given my interest in security I try to not make my own code vulnerable - even tough it might take more time to complete. You can also find me on several sites offering hacking challenges like HackQuest or HackThisSite.

Robotics

Robots are fun. Their blend of hardware and software make them unique. You can't solely focus on your programming - you have to think of your hardware and how to interact with it as well. Concepts like time or behaviour also play a strong role. The programming style is totally different and it feels more like you are actually creating something, when you finally get your robot to find through a labyrinth. I've built a simple line follower, a small segway, a labyrinth "solver" and a quadcopter. Nowadays I prefer an Arduino (or a Raspberry Pi for some applications), but I've also used a PIC and Lego NXT.

Artificial Intelligence

I care most about the more practical aspects of AI which includes topics like path planning, localization, mapping or machine learning. It's not that I completely dislike formal systems - on the contrary I fully understand their necessity and what they provide - I just like actually seeing something, more. It's fascinating if a robot is able to track its position within an environment or plan the shortest path to some goal. I also find the concept of machine learning quite thrilling - actually teaching some machine how to do something based on data, instead of writing a specific algorithm - who doesn't want that?!

Traveling

I'm a huge fan of travelling - not only because it's a fantastic opportunity for taking pictures but also because it's nice to leave the stressful working life behind me and experience new cultures. I'm particularly fascinated with Asia and Africa. Asia because of the tremendous cultural differences and the values of its society. Even though Asia has the fasting growing economy in the world, during my visits I've experienced a respect for traditions and a spirituality that I have yet to see in Austria. My interest in Africa on the other hand lies in its raw nature and wildlife. Visiting Africa has always been an adventure for me.

Movies

When I'm not coding something, I tend to watch movies and series. I usually prefer thrillers, adventure- or action-movies, but I'm always open to suggestions. I closely following TV series like Game of Thrones, Breaking Bad, Sherlock or Dexter. Given the sorry state of Austrian TV I usually get my movies from services like iTunes. I'm also a little bit of an audio fanatic - a fact my neighbors probably only know to well. But let's be honest - you just need surround sound for a cinematic experience...

Programming

Like every computer science student I tend to code a lot. I've dabbled in C, Haskell, Java, Python, Bash, SQL, PHP, Javascript and Octave (Matlab). The ones I've used the most are definitely Java and Python. I'm always interested in learning new stuff and I find things like Node.JS fascinating - sadly I haven't had the time to take a closer look yet. Given Java's slow development I'm currently learning Scala, as I think it's the most promising successor for a JVM-based language. The blend of object oriented and functional language elements make it really attractive. I also wonder if I will actually live to see Lambda expressions in Java.

Java

Both at university and at work Java is often the language of choice. I've developed both client and server applications in Java using different frameworks like Eclipse RCP, JSF, Lucene, Hibernate or Spring. I prefer web based three-tier architectures with REST. Although Java is often frowned upon, I actually think it is a good choice for many projects due to its simplicity, good library support and testability.

Automation

I get fed up with deploying applications and setting up development environments rather quickly. For this reason I use Vagrant with Puppet/Chef and container engines like Docker. Setting up a development database with Vagrant and distributing just the Vagrantfile to all the project members is so much easier than copying VMs or explaining everyone how to set up the database manually - which leads to people having different problems instead of just one common problem. I also think that for a successful agile development workflow, it is essential to use version control systems like Git and build servers.

Python

There are applications for which Java is not a good choice. Small scripts, complex algorithms or scientific use cases like A.I. or machine learning usually suffer from Java's verbose nature - which is why I use Python. Python's good readability and powerful nature make it an excellent choice for these problems. You can see some of my Python code here.

Quadcopter

I spent the summer of 2012 building my own quadcopter. I built both the hard- and software myself. It uses an Arduino microcontroller with a DCM-based sensor fusion coupled with a PID controller for stabilization. For wireless communication I am using XBee. I plan to eventually write up a complete tutorial once I've ironed out the kinks that caused the last crash. Programming the Arduino was a nice change of pace from all the business applications and required a more low-level approach.

My Background

I'm currently studying Software and Information Engineering at Vienna University of Technology, finishing my Bachelor in summer 2014 (knock on wood). Additionally I work as a part time software developer for Celix.

Prior to attending university I went to HTL Spengergasse - a secondary technical college for informatics. After three years I focused on network engineering and robotics. During my time there I completed the CCNA and realized both class accompanying and a final-year project. These included the migration of the school's telephone infrastructure to Cisco Unified Communications and the implementation of a virtualization infrastructure based on VMware, HP Blades/SAN and Wyse Thin Clients.

After finishing school I worked for two months as a system administrator managing a medium-sized network including Active Directory, Exchange, Cisco Blade/Stackable Switches and building a new Cisco Aironet WiFi with Radius authentication. Later that year I finished my six months military service at the IT security department of the Austrian Armed Forces.
